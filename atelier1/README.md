# Configuration de l'environnement

## Installation GIT

En premier lieu nous allons installer le client de versionning [GIT](https://git-scm.com/downloads)

Eventuellement vous pouvez utiliser un [client GUI](https://git-scm.com/downloads/guis) 

## Méthode 1:

1. Installation [JDK 8 ou plus](https://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html)
2. Installation [Maven](https://www-us.apache.org/dist/maven/maven-3/3.6.2/binaries/apache-maven-3.6.2-bin.zip)
3. Installation IDE [STS](https://spring.io/tools3/sts/all)
4. Installation de [Postman](https://www.getpostman.com/downloads/)

## Méthode 2:

1. Téléchargez [VirtualBox](https://www.virtualbox.org/wiki/Downloads)
2. Importer l'image VirtualBox contenant l'environnement de développement pré-configurer
